# Employee Task Tracker

A simple client-side web application built using HTML, CSS, and JavaScript to manage employee tasks.

## Features
- Add new tasks
- Delete tasks
- Persistent data storage using browser localStorage
- Clean and minimal UI

## Technologies Used
- HTML for structure
- CSS for basic styling
- JavaScript for DOM manipulation and logic

## How It Works
- Tasks are added dynamically using JavaScript
- The DOM is updated without page reloads
- Task data is stored in localStorage to maintain persistence across sessions

## How to Run
1. Clone the repository
2. Open `index.html` in any modern web browser
